# OpenAirSim

**OpenAirSim** is an open source airline simulator, covering from 1946 to 2035. It provides dynamic economy, real aircraft and airports, and historic events that affect your operations.

---

## Technologies used

- **C# (.NET 8)**
- **MIT License**
